## Features
- **Premium Glassmorphism**: High-quality frost effects and subtle borders.
- **Dynamic Backgrounds**: Floating blobs that respond to mouse movement.
- **Responsive Grid**: Works perfectly on mobile, tablet, and desktop.
- **Allumnova Branding**: Integrated navy and orange color palette.
- **Micro-interactions**: Smooth scroll reveals and hover state animations.

## Usage
Simply clone the repository and open `index.html`, or copy the CSS snippets from the live demo.

```bash
git clone https://github.com/allumnova/auraui.git
```
